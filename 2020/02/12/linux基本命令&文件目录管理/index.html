3.cat命令

cat命令用于查看文件内容，经常和grep命令连用

4.ifconfig命令

ifconfig命令用来查看网卡信息，如果命令找不到，则可直接运行yum -y install net-tools即可

5.hostname命令

hostname命令用来查看主机名或更改主机名（更改重启后生效）

6.su命令

su命令用来切换用户，需要注意的是：普通用户切换到root用户需要验证密码，而root用户切换到普通用户不需要验证密码。
