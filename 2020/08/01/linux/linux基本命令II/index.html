linux基本命令

Linux
Linux基础
发布日期:   2020-08-01

一、cat命令

1.cat用途

​ 显示文件内容

2.命令格式

​ cat [选项] 文件名

3.常用选项

  1. -b：非空行加上序号
  2. -A：show all ==-vET
  3. -v：–show-nonprinting，使用^和M-引用，除LFD和TAB
  4. -E：–show-ends，在每行结束处显示“$”
  5. -T：–show-tabs，将跳格字符显示为^I

二、tac命令

1.tac用途

​ 倒叙浏览，适用于查看系统日志等按时间更新的文件

2.命令格式

​ tac [选项] 文件名

3.常用选项

  1. -b：before 在行前而非行尾添加分隔标志
  2. -r：regex 将分隔标志视作正则表达式来解析
  3. -s：separator 使用指定字符串代替换行符做分隔标志

三、more&less命令

1.more用途

​ 全屏方式分页显示文件内容

2.more命令格式

​ more [选项] 文件名

3.more操作方法

  1. enter：向下逐行滚动
  2. space：向下逐页翻动
  3. q：退出

4.less用途

​ 与more功能相同，但扩展功能更多

5.less命令格式

​ less [选项] 文件名

6.less操作方法

  1. PgUP ：上翻页
  2. PgDn ：下翻页
  3. ‘/’键查找内容，‘n’下一个内容，‘N’上一个内容

四、head&tail 命令

1、head

head：查看文件开头的前十行（默认）

选项：-n ：n代表行数

2、tail

tail：查看文件末尾的后十行（默认）

选项：

  1. -n：n代表行数

  2. -f ：跟踪文件末尾的内容变化，实时显示更新的内容

五、wc命令

1.wc（Word count）

​ 统计文件中单词数量等信息，输出结果依次为行数、单词数、字符数、文件名

2.命令格式

​ wc [选项] 目标文件

3.选项

  1. -l：line 统计行数
  2. -w：Word 统计单词个数
  3. -c：统计字节

六、grep命令

1.grep

​ 检索和过滤文件内容，在文件中查找显示包含指定字符串的行

2.命令格式

​ grep [选项] 查找条件 目标文件

3.常用选项

  1. -i：查找时忽略大小写
  2. -v：反转查找，取反，输出与查询条件不相符的行

4.查找条件

​ 要查找的字符串以英文双引号括起来

  1. ”^..”：表示以”..”开头
  2. “..$”：表示以”..”结尾
  3. “^$”：表示空行

七、压缩&解压

1.gzip命令

1.1.gzip：用于制作压缩文件，解开压缩文件（不保留原文件）

1.2.选项：

​ -9 ：表示高压缩比，用在创建压缩文件时使用

​ -d ：用于解压压缩文件

1.3.压缩：gzip -9 文件名

1.4.解压缩：gzip -d .gz格式的压缩文件

2.bzip2命令

2.1.bzip2：用于制作压缩文件，解开压缩文件，压缩效率更高（不保留原文件）

2.2.选项：

​ -9 ：表示高压缩比，用在创建压缩文件时使用

​ -d ：用于解压压缩文件

2.3.压缩：bzip2 -9 文件名

2.4.解压缩：bzip2 -d .bz格式的压缩文件

3.tar命令

3.1.tar：制作归档文件，释放归档文件，用来压缩文件夹（打包）（保留原文件）

3.2.打包：tar [选项] 归档文件名 源文件或目标文件

3.3.解包：tar [选项] 归档文件名 [-C 目标目录]

3.4选项

  1. -c ：create 创建.tar格式的包
  2. -x ：解开.tar格式的包
  3. -v ：输出过程
  4. -f ：表示使用归档文件（使用其它选项时必须加上-f选项连用）
  5. -p ：打包时保留原始文件及目录权限
  6. -t ：列表查看包内的文件
  7. -C ：解包时指定释放的目标文件夹
  8. -z ：调用gzip程序进行压缩或解压
  9. -j ：调用bzip2程序进行压缩或解压
