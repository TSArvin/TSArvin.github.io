循环中的关键字

continue和break都是用在循环语句中，都用来跳出循环，区别在于break用来跳出整个循环执行循环体之后的语句，而continue用来跳出本次循环继续下次循环，不执行当前循环中continue语句后面的循环体。

break

break语句可以跳出for和while的循环体，如果使用break来终止循环，任何对应的循环else块将不执行。（终止）

实例：

>>> n = 5
>>> while n >0:
    n -=1
    if n == 2:
        break
    print(n)

上述代码输出结果如下：

4
3

continue

continue语句被用来告诉python跳过当前循环块中的剩余语句，然后继续进行下一轮循环。（中止）

实例：

>>> n = 5
>>> while n >0:
    n -=1
    if n == 2:
        continue
    print(n)

上述代码输出结果如下：

4
3
1
0

else

循环语句可以有 else 子句，它在穷尽列表(以for循环)或条件变为 false (以while循环)导致循环终止时被执行，但循环被 break 终止时不执行。

实例：（判断十以内的质数）

for n in range(2, 10):
    for x in range(2, n):
        if n % x == 0:
            print(n, '等于', x, '*', n//x)
            break
    else:
        # 循环中没有找到元素
        print(n, ' 是质数')

执行上述代码块的输出结果如下：

2  是质数
3  是质数
4 等于 2 * 2
5  是质数
6 等于 2 * 3
7  是质数
8 等于 2 * 4
9 等于 3 * 3

pass

Python pass是空语句，是为了保持程序结构的完整性。pass 不做任何事情，一般用做占位语句。
